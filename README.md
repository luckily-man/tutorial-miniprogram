# 教程大全 · 微信小程序（云开发共享版）

一个**用户上传并查看图文 + 视频教程**的微信小程序。场景示例：怎么坐地铁、怎么坐飞机、怎么做某道菜……教程由用户上传（文字步骤 + 视频），**所有打开小程序的人都能看到**。

本工程基于**微信云开发（CloudBase）**：教程存云端数据库、视频存云存储，无需自建服务器，即可实现跨设备、多人共享。

---

## 一、功能

- 🏠 **首页**：搜索 + 分类（全部 / 地铁 / 飞机 / 做菜 / 其他）+ 教程卡片列表（来自云端，所有人可见）
- 📖 **详情页**：分步文字说明 + 视频播放（云存储视频免域名校验，真机可播）
- ➕ **上传页**：填标题 / 选分类 / 写简介 / 动态添加文字步骤 / 选择或拍摄视频（视频自动上传到云存储）
- 👤 **我的**：浏览全部教程，仅可删除**本机上传**的内容

数据存于微信云开发：数据库集合 `tutorials`（教程）+ 云存储目录 `tutorials/`（视频）。内置 4 条示例教程（地铁 / 飞机 / 番茄炒蛋 / 手冲咖啡），首次启动自动写入云端。

## 二、如何运行（重要：需 3 步前置配置）

### 1. 导入项目
1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
2. 打开开发者工具 →「导入项目」→ 目录选择本文件夹 `wechat-tutorial-miniprogram`。
3. AppID 填你自己的小程序 AppID（本工程已配好 `wx2dc03918a1bc422d`，如换了号自行修改 `project.config.json` 的 `appid`）。

### 2. 开通云开发并填入环境 ID
1. 工具顶部点「**云开发**」按钮 → 开通 → 新建一个环境（个人主体有免费额度）。
2. 进入「**环境设置**」复制你的**环境 ID**（形如 `cloud1-xxxxxxxx`）。
3. 打开 `app.js`，把顶部的占位常量替换成你的环境 ID：
   ```js
   // ⚠️ 把下面替换成你自己的「云开发环境 ID」
   const CLOUD_ENV = 'cloud1-xxxxxxxx'   // ← 改成你的
   ```
   > 若未填写，编译时会弹窗提示「未配置云环境」，属正常引导。

### 3. 创建数据库集合并设置权限（关键）
1. 在「云开发」控制台左侧点「**数据库**」→「新建集合」，名称填 **`tutorials`**。
2. 进入该集合 →「**权限设置**」→ 选择 **「所有用户可读，仅创建者可读写」**。
   - ⚠️ 这一步决定「**所有人都能看**」是否成立。默认权限「仅创建者可读写」会导致其他人看不到教程！
3. （可选）「**存储**」→ 确认 `tutorials/` 目录存在；云存储默认「所有用户可读」，视频人人可播。如看不到视频，可在存储权限里设为「所有用户可读」。

### 4. 编译运行
点击「编译」，模拟器即可看到教程列表；点「预览」扫码即可在手机上查看，**别人手机打开也能看到你上传的教程**。

## 四、工作原理

- **共享**：教程写入云数据库集合 `tutorials`，权限设为「所有用户可读」，因此任何打开小程序的人都能在首页/详情看到。
- **视频**：上传时 `wx.cloud.uploadFile` 存到云存储，返回 `fileID`（`cloud://...`）写入记录；`<video>` 直接用 `fileID` 播放，免域名校验，真机可播。
- **身份与删除**：云开发写入记录自动带 `_openid`，但前端不可读。本工程用本机标记（`my_upload_ids`）判断「我上传的」，仅这些显示删除按钮。删除走云权限「仅创建者可写」，删他人会失败并提示。
- **示例数据**：`app.js` 启动时若云端集合为空，自动把 `seed.js` 的 4 条写入（仅一次）。

## 五、可扩展方向

- 登录与作者体系：用云函数返回 `openid`，让「我的」跨设备一致（替代本机标记）。
- 点赞 / 收藏 / 评论（新增云集合）。
- 教程封面图、步骤配图（上传云存储）。
- 教程审核流（避免违规内容）。
- 分享到微信群 / 朋友圈（`onShareAppMessage`）。

---

## 六、回退到「本地存储版」（不需要云开发时）

若暂时不想用云开发，可把 `utils/store.js` 与 `app.js` 改回本地存储实现（历史版本即用 `wx.setStorage`）。本地版教程仅存本机，不共享。
